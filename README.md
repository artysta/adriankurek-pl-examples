# How to run?

1. Clone branch.

        `git clone --single-branch --branch java-keyadapter https://github.com/artysta/adriankurek-pl-examples.git`

2. Go to the folder.

        `cd adriankurek-pl-examples/java-keyadapter/`

3. Compile.

        `javac pl/adriankurek/App.java`

4. Run.

        `java pl.adriankurek.App`
